# node.js + socket.io 群聊系统

主要功能:
普通用户可以在群里聊天
管理员登录后可以在群里聊天, 也可以踢人

详细功能:
收到消息有声音提醒  
进入聊天时会显示最近的聊天记录  
保存每一天的聊天记录到"日期.txt"文本中  
可以更换聊天背景  
实时显示在线人数  
实时显示聊天人员变动情况  
实时显示谁在输入消息   
管理员输入的字体颜色不同  
敏感字替换 (例如: "我是你爸爸"会替换成"我是你儿子"等等... )  
  
  
项目已经部署到了云服务器, [查看效果请戳我](http://120.79.217.212:4000/) (2019年10月底服务器过期)
